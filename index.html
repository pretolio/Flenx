Começar

Instale e rode em 4 passos

1 · Instalar  •  2 · Gerar entrypoints  •  3 · Rodar

dart pub add flenx jaspr
dart pub add dev:build_runner dev:build_web_compilers dev:jaspr_builder

dart run flenx:bootstrap          # gera os entrypoints do jaspr
dart pub global activate jaspr_cli
jaspr serve                       # http://localhost:8080 (hot reload)

Os exemplos são SSR — rode localmente com jaspr serve na pasta de cada um.
